<Steps>
  <Step title="Загрузка приложения">
    Скачайте установщик приложения Hiddify по ссылке -\> [Скачать](https://github.com/hiddify/hiddify-next/releases/download/v2.0.5/Hiddify-Windows-Setup-x64.exe)
  </Step>
  <Step title="Установка приложения">
    Запустите установщик, пройдите шаги:

    1. _Язык установки:_`Русский`
    2. _Создать ярлык на рабочем столе:_ <Icon icon="check" iconType="solid" />
    3. _Автоматически запускать Hiddify:_ <Icon icon="check" iconType="solid" />
    4. `Далее` -\> `Установить`
    5. _Запусть Hiddify:_ <Icon icon="xmark" iconType="solid" color="#fa3f3f" />
  </Step>
  <Step title="Настройка приложения">
    Перед первым открытием приложения нажмите **ПКМ** по ярлыку **Hiddify** на рабочем столе -\> `Свойства` -\> `Ярлык` -\> `Дополнительно` -\> `Запускать от имени администратора` <Icon icon="check" iconType="solid" />

    При первом открытии **Hiddify** попросит настроить язык, регион и сбор аналитики

    <Warning>
      Обязательно установите эти параметры:

      Регион: `Другой`

      Сбор аналитики: `Выкл`
    </Warning>
    ![Windows Hiddify Setup Pn](/images/windows/Windows_Hiddify_Setup.png)

    **Hiddify** может работать в двух режимах:

    - **Системный прокси (по умолчанию)** - Использовать VPN будут только браузеры и некоторые другие приложения, поддерживающие работу через прокси. Все остальные программы будут работать напрямую, без VPN.
    - **VPN** - Абсолютно весь ваш интернет-трафик будет идти через VPN, вне зависимости от программ.

    <Note>
      Выберите тот режим, который соответствует вашим потребностям. Изменить режим работы можно в пару кликов:

      `Главная` -\> <Icon icon="sliders-simple" /> _(в правом верхнем углу)_
    </Note>
  </Step>
  <Step title="Добавление подписки">
    1. Скопируйте ссылку на подписку из [HoundVPN Bot](https://t.me/HoundVPN_Bot)
    2. Откройте приложение **Hiddify**
    3. Нажмите `＋` в правом верхнем углу -\> Добавить из буфера обмена
  </Step>
</Steps>
