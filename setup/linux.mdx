### NekoRay

<Steps>
  <Step title="Загрузка приложения" icon="cloud-arrow-down" iconType="solid">
    Перейдите по [ссылке](https://github.com/MatsuriDayo/nekoray/releases), спуститесь к разделу **«Assets»** актуальной версии программы и скачайте **file_name.deb** файл.
  </Step>
  <Step title="Установка приложения" icon="floppy-disk" iconType="solid">
    Перейдите в директорию с загруженным пакетом и выполните команду:

    ```bash
    sudo dpkg -i file_name.deb
    ```

    <Note>
      Не забудьте изменить **file_name** на актуальное название пакета.

      На момент написания инструкции это **«nekoray-4.0.1-2024-12-12-debian-x64.deb»**
    </Note>
  </Step>
  <Step title="Настройка приложения" icon="gear" iconType="solid">
    <Warning>
      Если запуск программы из меню приводит к её неработоспособности, выполните запуск через **Command Launcher** (ALT\+F2):
    </Warning>
    ```bash
      /opt/nekoray/nekoray
    ```

    При первом запуске программы выберите поддержку `Sing-Box`. Название программы изменится на `NekoBox`. Для настройки автозапуска перейдите во вкладку `Program` активируйте **«Start with system»**.
  </Step>
  <Step title="Добавление подписки" icon="plus" iconType="solid">
    1. Скопируйте ссылку на подписку из [HoundVPN Bot](https://t.me/HoundVPN_Bot)
    2. Перейдите во вкладку `Servers` **-\> «Add profile from clipboard»**
    3. 
  </Step>
</Steps>
